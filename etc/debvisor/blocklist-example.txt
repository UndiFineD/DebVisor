# DebVisor Blocklist Example
#
# This file demonstrates the format for blocklist entries used by DebVisor
# for network security filtering and access control.
#
# Format: One entry per line - CIDR blocks or individual IP addresses
# Supported formats:
#   - IPv4 CIDR: 192.168.1.0/24
#   - IPv4 single IP: 10.0.0.1
#   - IPv6 CIDR: 2001:db8::/32
#   - IPv6 single IP: 2001:db8::1
#
# Lines starting with '#' are comments and are ignored
# Inline comments are supported: 10.0.0.0/8 # Private RFC1918 range
# Blank lines are ignored
#
# Validation: Run './validate-blocklists.sh --blocklist blocklist-example.txt'
# to verify syntax and detect overlaps with whitelists
#
# USAGE: To activate this blocklist in your Ansible inventory:
#   debvisor_blocklist_sources:
#     - /etc/debvisor/blocklist-example.txt
#
# CAUTION: Review all entries carefully before deploying!
# Incorrect entries may disrupt critical system connectivity.
